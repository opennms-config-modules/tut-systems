= Tut-Systems

OpenNMS configuration for Tut-Systems devices.

== Installation

.Download and install from GitHub
[source, bash]
----
mkdir ${HOME}/opennms-config-workspace
cd ${HOME}/opennms-config-workspace
git clone https://github.com/opennms-config-modules/tut-systems.git
----

=== Events

.Install events
[source, bash]
----
cp ${HOME}/opennms-config-workspace/tut-systems/events/*.xml ${OPENNMS_HOME}/etc/events
----

.Include in eventconf.xml
[source, xml]
----
<event-file>events/tut-systems.events.xml</event-file>
----
